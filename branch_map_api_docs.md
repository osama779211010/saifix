# توثيق API مواقع فروع الوكلاء (Branch Map API)

يوفر هذا الـ API قائمة بجميع فروع الوكلاء النشطة التي تمتلك إحداثيات جغرافية (خطوط الطول والعرض)، ليتم عرضها كدبابيس (Markers) على الخريطة في التطبيق.

## معلومات الـ Endpoint

- **URL:** `/api/core/map-branches/`
- **Method:** `GET`
- **Authentication:** `Token` (مطلوب تسجيل الدخول)

## تعليمات التنفيذ لمبرمج التطبيق

### 1. عرض الدبابيس (Markers) على الخريطة
يمكنك استخدام مكتبة `Leaflet` (كما فعلنا في نسخة الـ Desktop) أو أي مكتبة خرائط أخرى في Flutter/React Native. المنطق البرمجي لإضافة الدبابيس هو:

```javascript
// مثال باستخدام Leaflet
branches.forEach(branch => {
    if (branch.lat && branch.lng) {
        L.marker([branch.lat, branch.lng])
            .addTo(map)
            .bindPopup(`<b>${branch.name}</b><br>${branch.agent_name}`);
    }
});
```

### 2. عرض تفاصيل الفرع عند الضغط
بدلاً من مجرد إظهار Popup صغير، نوصي بتطوير **بطاقة معلومات (Info Card)** تظهر في أسفل الشاشة (Bottom Sheet) عند الضغط على أي دبوس.

**المعلومات التي يجب عرضها في البطاقة:**
- اسم الفرع + اسم الوكيل التابع له.
- العنوان الكامل (المحافظة - المنطقة - الشارع).
- أرقام التواصل (جوال + ثابت).
- زر "الانتقال إلى الخريطة" يفتح رابط `map_url`.

### 3. ميزة البحث والتصفية
بما أن الـ API يعيد كافة البيانات، يمكنك عمل تصفية محلية (Local Search) في التطبيق للبحث عن فرع معين بالاسم أو المحافظة لتسهيل الوصول إليه على الخريطة.
